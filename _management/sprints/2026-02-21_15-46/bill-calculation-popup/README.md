# Sprint — Bill Calculation Popup

Sprint Name: bill-calculation-popup  
Sprint Owner: Michael  
Status: OPEN

## Sprint Goal

Add a bill calculation feature accessible from the top-left button. It opens a popup that lets the user select a table from available open tables, then shows a full bill breakdown including EXTRA orders.

## Scope

Included:
- Top-left “Bill” button
- Popup flow
- Table picker from available open tables
- Bill breakdown including extras
- Totals calculation

Excluded:
- Payments
- Printing
- Discounts/tips (unless explicitly requested)

## Deliverables
- Button visible on Waiter UI (top-left)
- Popup opens reliably on mobile/desktop
- Table selection list populated from DB data
- Bill includes base + all extras for that table
- Correct totals displayed
